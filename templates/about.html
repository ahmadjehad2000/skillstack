About Skillstack

What is Skillstack?

Skillstack is a powerful web application that uses advanced AI technology to generate complete, ready-to-teach courses on any topic. Our platform provides educators, trainers, and content creators with a tool to rapidly develop professional educational content with minimal effort.

How It Works

  1. Input Your Topic: Simply enter the subject you want to teach, select the difficulty level, and provide your API key.
  2. AI-Powered Generation: Our advanced AI system creates a comprehensive course with detailed content that covers all aspects of your topic.
  3. Custom Thumbnail: An eye-catching thumbnail is generated that represents your course topic.
  4. Detailed Content: You get a single comprehensive lesson covering fundamental concepts, methodology, exercises, and assessments.
  5. Ready to Use: Download your complete course in JSON format, ready to import into your learning management system.

Frequently Asked Questions

Skillstack is free to use, but it requires your API key to generate content. You will be charged based on your API usage. Typically, a complete course generation costs between $0.50 and $3.00 in API credits, depending on the course complexity and length.

We do not store your API key. It is used only for the current request and is not saved on our servers. For added security, we recommend using an API key with usage limits set in your dashboard.

Currently, we support exporting your courses in JSON format, which can be easily parsed and imported into various learning management systems. We plan to add more export formats in the future, including HTML, PDF, and direct LMS integrations.

Currently, the web interface does not support direct editing. However, you can download the course JSON and modify it as needed. We recommend using the generated content as a solid foundation that you can then customize and enhance based on your specific requirements.
